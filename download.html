Install notes

  • Windows: Run evolve-v4.1.8-windows-x64-setup.exe. Verify the SHA-256 checksum before installing. Windows SmartScreen may ask you to confirm until Authenticode-signed releases ship.
  • Android: Download evolve-v4.1.8-android-setup.apk from GitHub Releases (not a gh-pages binary path). Requires Android 7.0+ (API 24). If an older Evolve install was debug-signed, uninstall it first, then install this release-key APK. Enable “Install unknown apps” for your browser/files app. APK is signed with the Evolve release key (not debug). Camera permission is requested only when you scan a PERC QR code.
  • macOS: Download evolve-v4.1.8-macos-x64.zip, unzip, and open Evolve.app. Signed with Developer ID Application (Team SFCBP95595). Gatekeeper may still require a right-click → Open on first launch until notarization is applied.
  • iOS: Install evolve-v4.1.8-ios-setup.ipa via sideload tooling (AltStore, Apple Configurator, or MDM). Development-signed with Apple Development (Team SFCBP95595); not an App Store / TestFlight distribution IPA. Requires iOS 13+. Face ID is used for wallet unlock when enabled.
  • Web: Hosted at the same GitHub Pages site. Grok construal uses local heuristic mode in the browser.
  • Integrity: Evolve Android APK is release-key signed. macOS zip is Developer ID signed. Windows installers are checksum-verified (not yet Authenticode-signed). SHA-256 / SHA-512 sidecars on Evolve Releases. Verify with CHECKSUMS.sha256 or CHECKSUMS.sha512 (or checksums.json).
  • Updates: Check GitHub Releases for older versions and release notes.
