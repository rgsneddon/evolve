Install notes

  • Windows: Run evolve-v4.1.3-windows-x64-setup.exe. Verify the SHA-256 checksum before installing. Windows SmartScreen may ask you to confirm.
  • Android: Run evolve-v4.1.3-android-setup.apk. Requires Android 6.0+ (API 23). Verify SHA-256 before installing. Camera permission is requested only when you scan a PERC QR code.
  • iOS: Install evolve-v4.1.3-ios-setup.ipa via sideload tooling (AltStore, Apple Configurator, or MDM). Requires iOS 13+. Face ID is used for wallet unlock when enabled.
  • Web: Hosted at the same GitHub Pages site. Grok construal uses local heuristic mode in the browser.
  • Integrity: Every package ships with SHA-256 and SHA-512 checksum sidecars (not Authenticode-signed). Verify with CHECKSUMS.sha256 or CHECKSUMS.sha512 (or checksums.json).
  • Updates: Check GitHub Releases for older versions and release notes.
