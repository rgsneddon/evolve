Install notes

  • Windows (Evolve): Run evolve-v4.1.5-windows-x64-setup.exe. Verify the SHA-256 checksum before installing. Windows packages are Authenticode-signed for a trusted install path. The installer bundles WireGuard and a VPN profile — no separate WireGuard install required.
  • Android (Evolve): Run evolve-v4.1.5-android-setup.apk. Requires Android 6.0+ (API 23). APK is signed with the Evolve release key (not debug). Verify SHA-256 before installing. Camera permission is requested only when you scan a PERC QR code.
  • MY PERC macOS / iOS: Use the MY PERC (Perccent Wallet) section above — macOS is the first card. Full list: MY PERC downloads. There is no Evolve Chronoflux iOS platform installer.
  • Web: Hosted at the same GitHub Pages site. Grok construal uses local heuristic mode in the browser.
  • Integrity: Evolve packages are Authenticode-signed (Windows) and release-key signed (Android), with SHA-256 / SHA-512 sidecars on Evolve Releases. MY PERC: perccent-wallet v1.1.6.
  • Updates: Evolve: GitHub Releases. MY PERC: perccent-wallet releases.
